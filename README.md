# Lab2_News-Client
Новостной клиент на Android с использованием Retrofit, Coroutines, Coil и Material Design. Позволяет просматривать актуальные новости через бесплатный API и искать по ключевым словам и открывать полные статьи в браузере. Показывает их в виде прокручиваемой ленты с форматированным выводом

Возможности:
- Лента новостей отображает главные заголовки со всего мира (GNews API: https://docs.gnews.io/)
- Быстрый поиск новостей по ключевым словам
- Интуитивно привычное обновление ленты свайпом вниз
- Дает возможность перейти к полной статье в браузере при нажатии на карточку
- Есть понятные сообщения при отсутствии сети или данных
- Фильтр по категориям (general, world, business, technology, sports, science, health)
